Above & Beyond

Concept


We wanted to create a connected and valuable experience. We brainstormed heavily when coming up with a concept which could both engage and educate young people. We strived to create a seamless experience and a journey like feeling.


Design


Inspired by flat design, we came up with our design style. We wanted to use bright colors and bold iconography to enhance the visual aspect of the project. This also helped create a friendly and welcoming website experience. The scroll interaction was very important and crucial for the project as we wanted the user to feel like they were on a journey moving through the site.


Code


Animated Navigation: Navigation hides when user scrolls down. It animates in when the user scrolls up again.


Scroll To top arrow: A small arrow fades in after the user scrolls a certain height of the page. On clicking that arrow the pages scrolls back up to the top.


Hamburger Menu: Animated hamburger menu for mobile view.
